# Magic Mic

Welcome to Magic Mic's Baseball Card Collection & Store. This space was created from a genuine love for the game and the fun of collecting. It started as a personal hobby and slowly grew into an online shop where collectors, fans, and anyone curious can find cards worth keeping. Whether you are looking for something rare or just exploring, the goal is to share the excitement and history of baseball through every card in the collection.

## Live demo

https://magic-mic.vercel.app

## Build with

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **React Router** - Navigation
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
